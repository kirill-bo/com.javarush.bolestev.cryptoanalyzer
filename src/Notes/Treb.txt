Техническое задание
Разработать программу, позволяющую шифровать и расшифровывать текст с использованием шифра Цезаря.

Программа должна поддерживать несколько режимов работы:
1) Шифровка текста +
2) Расшифровка текста с помощью ключа +
3) Расшифровка текста с помощью brute force (перебор всех вариантов) +

- Программа должна открывать указанный файл с текстом +
- Проделывать с ним одно из указанных выше действий +
- После этого создать новый файл с результатом +
- Обеспечивать обработку больших файлов +
- Иметь валидацию входных данных +

Опционально:
- Добавить графический интерфейс пользователя
- Статический анализ для автоматического взлома шифра +

Классы:
1. MainApp: Это главный класс, откуда начинается выполнение программы.
Отвечает за обработку команд пользователя, вызов соответствующих методов и управление потоком работы программы.
2. Cipher: Класс, реализующий функциональность шифра Цезаря и дешифровки.
3. FileManager: Отвечает за чтение и запись файлов.
4. Validator: Валидация входных данных, таких как существование файла, допустимость ключа.
5. BruteForce: Реализация метода перебора всех ключей для взлома.
6. StatisticalAnalyzer: Для статистического анализа при расшифровке.
